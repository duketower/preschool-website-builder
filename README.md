# Preschool Website Builder

A complete system to build conversion-focused preschool websites for Indian schools.

## What's Included

| Folder | Contents |
|---|---|
| `research/` | Indian preschool UX & conversion analysis |
| `shared/` | CSS design system, JS skills, reusable components |
| `templates/preschool-site/` | Reusable 9-page website template |
| `templates/pitch-decks/` | Internal sales deck + client-facing deck (A4 printable) |
| `demo-site/` | Fully populated demo site — Sunshine Sprouts Preschool |
| `docs/` | Usage guide, rebranding instructions, deployment help |

## Quick Start

1. Open `demo-site/pages/index.html` in your browser to see the demo site
2. Open `templates/pitch-decks/internal-sales-deck/index.html` for the sales deck
3. Open `templates/pitch-decks/client-deck/index.html` for the client deck
4. Read `docs/usage-guide.md` to learn how to rebrand the template for a new school

## Tech Stack

- Vanilla HTML, CSS, JavaScript — no build tools required
- Google Fonts (Poppins + Nunito)
- Font Awesome 6
- Skills library: shared/js/
